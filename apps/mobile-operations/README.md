# Tarhib Operations

Internal Tarhib operations mobile app.

- Accent: Web Admin blue pastel.
- Scope: manager dashboard, order queue, preparation, delivery, stock and profile.
- Bundle IDs: `com.tarhib.operations`.

## Run

```powershell
cd C:\Users\A\Pictures\tarhib
npm install -w apps/mobile-operations
cd apps/mobile-operations
npm run web
```
