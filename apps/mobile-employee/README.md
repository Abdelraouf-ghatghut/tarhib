# Tarhib Employee

Client employee mobile app.

- Accent: green pastel.
- Scope: catalog, favorites, cart, order tracking, meeting rooms and profile.
- Bundle IDs: `com.tarhib.employee`.

## Run

```powershell
cd C:\Users\A\Pictures\tarhib
npm install -w apps/mobile-employee
cd apps/mobile-employee
npm run web
```
